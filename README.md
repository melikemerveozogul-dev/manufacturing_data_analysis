# Manufacturing Data Analysis Project 📊

This project analyzes a production dataset (120,000+ rows) to visualize equipment downtimes and calculate efficiency metrics within the scope of **Industry 4.0** and **MES**.

## Features
- **Data Processing:** Large datasets handled with Python & Pandas.
- **Visual Reporting:** Categorized downtime analysis using Matplotlib.
- **Efficiency Metrics:** Automated calculation of production yield and error rates.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Openpyxl (for Excel exporting)

## How to Run
1. Install dependencies: `pip install pandas matplotlib openpyxl`
2. Run the script: `python manufacturing_data_analysis.py`
